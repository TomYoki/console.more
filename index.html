Example of Console.more
Command+Option+J

Control+Shift+J

Console.image

console.image will load image,
get it's size and log it as if it was text.
It takes 1 input and doesn't return anything.
On Error, it will log error in console.

          
            console.image("https://suchimages.com/notacat.jpg");
            // This will load and log the image from specified resource.

Console.styles

console.styles takes 1 or more objects,
which it will remember across files.
You can also set such values as "default",
which will be used when calling `console.text` without 2nd parameter.
If you wish to unset all style values,
call `console.style` without any parameters.

          
            console.styles({
              // This will act as "default" value for console.text();
              default: [
                "color: gray;",
                "font-size: 15px;"
              ],
              // This will be saved in console.more.styles as "underline".
              underline: [
                "text-decoration: underline"
              ]
            });

Console.text

Console.text takes 1 or 2 parameters,
depending if "default" value is set in `console.style`.
It doesn't return any value. 

          
            console.text("https://google.com/picture_of_a_cat.png", [
              "color: gray;",
              "font-size: 72px;"
            ]);
            //This will log text with specified properties

Check for compatibility

console.more.compatible will hold boolean value.
It will be true/false,
depending if browser supports custom logs.

          
            if(console.more.compatible){
              console.log("I am compatible with this browser!");
            } else {
              console.log("Aww :(");
            }
